Perguntas Frequentes para Parceiros

Quem pode se inscrever no Programa de Parceiros?

Qualquer pessoa maior de 18 anos com vontade de vender cursos online.

Como recebo o pagamento das comissões?

Os pagamentos são feitos via transferência bancária todo mês.

Preciso emitir nota fiscal por cada venda?

Não é necessário, a plataforma já cuida de toda a parte fiscal.

Qual é a média de conversão dos afiliados CED?

Temos afiliados com taxas acima de 5% de conversão.

Como acessar os materiais de divulgação?

Após o cadastro, enviamos o link com todos os materiais.
